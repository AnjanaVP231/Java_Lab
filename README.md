# ☕ Java Lab Programs

This repository contains simple Java programs for beginners.  
These programs cover fundamental concepts like condition checking, loops, arithmetic operations, and user input using Scanner.

---

## 📌 Programs Included

1. 🔢 Largest of Three Numbers  
2. 🔄 Swap Two Numbers (With and Without Third Variable)  
3. ⚖️ Check Whether a Number is Odd or Even  
4. ➕ Sum of Odd and Even Numbers up to a Given Range  
5. 🧮 Factorial of a Number
